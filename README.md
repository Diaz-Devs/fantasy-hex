# Fantasy Catan

This project was bootstrapped with the latest Vite React starter. Use the commands below to run and develop the app locally.

## Available Scripts

- `npm run dev` – Start the development server with hot module replacement.
- `npm run build` – Create a production build in the `dist` folder.
- `npm run preview` – Preview the production build locally.
- `npm run lint` – Lint the project files with ESLint.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the printed local URL to view the app.
